🌐SkillMitra – Local Skill Exchange Platform

📌About the Project:--
SkillMitra is a web-based platform that allows people to connect and exchange skills within a local community. Users can offer their skills and learn new ones from others, creating a collaborative learning environment.
This project is designed to encourage peer-to-peer learning and make skill-sharing accessible to everyone.

🎓Project Background:--
This is my college project
This project was built with the help of GPT tools
This is my first complete web development project

💡Features:--
User registration and login system
Add and showcase skills
Browse available skills from others
Connect with people for skill exchange
Simple and user-friendly interface

🛠️Technologies Used:--
Frontend: HTML, CSS
Backend: Python (Django)
Database: SQLite (default Django DB)
